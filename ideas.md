## Idea A: "Terminal Athlete" — Dark Brutalism + Developer Aesthetic

**Design Movement:** Brutalist Web + Terminal UI

**Core Principles:**
- Raw, unapologetic typography — heavy condensed headlines that feel like they were compiled, not designed
- Developer aesthetic as the primary visual language, not a gimmick
- High contrast: near-black backgrounds, pure white type, orange as the only color
- Sections feel like code blocks and terminal windows, but with athletic energy

**Color Philosophy:**
- Background: #111214 (near-black, deeper than current draft)
- Surface: #1e2023 (card backgrounds)
- Orange: #ff8200 (CPZ brand — used sparingly for maximum impact)
- White: #f5f5f5 (headlines)
- Code green: #22c55e (terminal text only)
- The orange should feel like a warning light — you notice it immediately

**Layout Paradigm:**
- Asymmetric split layouts — text-heavy left column, image/visual right
- Full-bleed hero with Philip's photo cut out against the dark background
- Horizontal scroll sections for the Pipeline stages
- Cards with hard borders (1px solid orange) instead of rounded shadows

**Signature Elements:**
- Terminal window chrome (red/yellow/green dots) on key callout cards
- Git log timeline in the About section (already in draft — keep and enhance)
- Orange left-border accent lines on section headers

**Interaction Philosophy:**
- Hover states reveal orange underlines and borders
- CTA buttons have a "compile" animation on hover (text shifts slightly)
- Scroll-triggered fade-ins that feel like code loading

**Animation:**
- Subtle typewriter effect on hero headline
- Section transitions: fade-up on scroll
- Pipeline cards: stagger-in from left

**Typography System:**
- Headlines: Space Grotesk Bold / Barlow Condensed ExtraBold (athletic + technical)
- Body: Inter Regular 16px
- Code: JetBrains Mono (terminal elements only)

## Idea B: "Spartan Engineer" — Bold Athletic + Minimal Code Accents

**Design Movement:** Modern Athletic Minimalism (inspired by Strivon reference)

**Core Principles:**
- Philip's physique is the hero — large, confident photography dominates
- Bold, condensed type that commands attention (like a Spartan race banner)
- Code/developer accents are present but secondary — the body speaks first
- Clean section transitions with diagonal cuts and strong contrast blocks

**Color Philosophy:**
- Background: #0f1012 (deep charcoal, almost black)
- Orange: #ff8200 (CPZ brand — primary CTA, section dividers, highlights)
- Off-white: #f0ede8 (warm white for body text — feels premium, not clinical)
- Mid-grey: #2a2d31 (card backgrounds, secondary sections)
- The palette is intentionally limited — 3 colors max per section

**Layout Paradigm:**
- Full-viewport hero with Philip's photo as the dominant element
- Diagonal section dividers (clip-path) to create visual momentum
- Stats bar as a horizontal band between hero and about section
- Pipeline as a numbered card grid (not accordion)
- Sticky nav with orange CTA always visible

**Signature Elements:**
- Diagonal orange slash dividers between sections
- Large numerals (01, 02, 03) as section anchors
- Philip's photo with a subtle orange color-grade overlay

**Interaction Philosophy:**
- Orange fills in on button hover (not just border)
- Cards lift with subtle shadow on hover
- Nav CTA pulses gently to draw attention

**Animation:**
- Hero: Philip's photo fades in from slight blur
- Stats: Numbers count up on scroll-into-view
- Pipeline cards: stagger-in with slight upward motion

**Typography System:**
- Headlines: Barlow Condensed ExtraBold (all-caps for major headings)
- Subheadings: Space Grotesk SemiBold
- Body: Inter Regular
- Code accents: JetBrains Mono (small, muted — used for labels only)

## Idea C: "System Upgrade" — Editorial Dark + Orange Neon Accents

**Design Movement:** Editorial Dark Magazine + Cyberpunk Minimal

**Core Principles:**
- The site reads like a technical magazine, not a fitness landing page
- Large editorial photography, generous whitespace within dark backgrounds
- Orange used as a neon accent — glows, not just fills
- Every section has a distinct "chapter" feel with large section numbers

**Color Philosophy:**
- Background: #13151a (dark navy-black)
- Orange: #ff8200 with a subtle glow (box-shadow: 0 0 20px rgba(255,130,0,0.3))
- Light: #e8e6e1 (warm off-white)
- Accent surface: #1c1f26
- The orange glow creates a sense of energy and urgency

**Layout Paradigm:**
- Magazine-style: alternating full-bleed and contained sections
- Large pull-quotes as visual anchors
- The Pipeline section uses a horizontal timeline (not vertical accordion)
- Community section has a "dashboard preview" mockup of the Skool community

**Signature Elements:**
- Orange neon glow on primary CTAs and key stats
- Large background text (e.g., "GEEK" in 20vw opacity-5 text behind sections)
- Horizontal rule with orange gradient as section separators

**Interaction Philosophy:**
- Buttons have an orange glow pulse on hover
- Section numbers animate in on scroll
- Parallax on hero photo

**Animation:**
- Hero: Parallax scroll on Philip's photo
- Stats: Animated count-up with orange glow
- CTA buttons: Glow pulse on hover

**Typography System:**
- Headlines: Bebas Neue (bold, condensed, athletic)
- Subheadings: Space Grotesk Medium
- Body: Inter Regular
- Code: JetBrains Mono

## Selected Design: Idea B — "Spartan Engineer"

This approach best serves the conversion goals (book a call, join community) while honoring the brand. Philip's physique becomes the hero element, creating immediate credibility. The bold athletic typography commands attention like the Strivon reference, while the developer code accents maintain the brand's unique positioning. The diagonal section dividers create visual momentum that guides the eye toward CTAs.
